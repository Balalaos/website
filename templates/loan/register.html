Description

This is a registration form for establishing a new loan. The application checks that all information entered correctly. If something is entered wrongly, the application tells you what the error is. After register the Show loans page will be shown to user.
    App follows this rules during registration:
  • The ID must have 7 digits.
  • It is possible to use both letters and numbers for the name.
  • Loan can not be added for blacklisted borrower
  • App checks if an ID in use by another borrower. If same ID and borrower are exist in database, the new loan will be added for this borrower.
  • The date should be minimum of 90 days after today day.
  • The app does not allow you to make more than 1 loan request in 24 hours.
