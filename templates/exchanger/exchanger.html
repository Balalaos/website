Description

This is an application for converting money from one currency to another. The transfer takes place through intermediary currency. The application's task is to find which intermediary currency is the most profitable for the user.

Also, after transferring money to the final currency, the final result is rounded down to the nearest cent. All excess remains as a profit for the company. The application also finds the best profit for the company.

To obtain data on current exchange rates, the API from SWEDBANK is used.
